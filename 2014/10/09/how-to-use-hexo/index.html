使用 hexo

1. 安装

  1. 安装node.js : http://nodejs.org/#download

  2. 安装npm:

    • git clone —recursive git://github.com/isaacs/npm.git
    • cd npm
    • sudo node cli.js install npm -gf

  3. 安装hexo: npm install hexo -g

  4. git clone https://github.com/yuqiao/yuqiao.github.io.git
  5. cd yuqiao.github.io
  6. hexo init
  7. npm install

  8. vi _config.yml # 配置git deploy

    deploy:

    type: github
repository: https://github.com/yuqiao/yuqiao.github.io.git
branch: master

  9. hexo generate

  10. hexo deploy

2. 使用

  1. 修改_config.yml
  2. hexo new “xxxx”
